2015-11-19 使用7牛图片存储

  • Github 空间有1G大小限制，另外Github服务器在国外, 考虑速度，将图片进行分流。

  • 目前尝试使用 7牛 www.qiniu.com 的图片存储。7牛免费资源 (10GB永久免费存储空间/每月10GB下载流量/每月10万次Put请求/每月100万次Get请求) 目前应该是够用了。

  • 关于图片的发布，7牛的同步工具 qrsbox 也挺好用的，可以自动监控同步文件图片，上传到服务器。7牛服务器没有目录，同步后链接前缀+图片文件名就可以直接访问了。

  • 关于图片的使用，赞一个的是7牛提供的图片外链支持参数，可以调整图片大小，弥补了Markdown不能定义图片大小的问题。

七牛图片基本处理（imageView2）
http://developer.qiniu.com/docs/v6/api/reference/fop/image/imageview2.html

    http://developer.qiniu.com/resource/gogopher.jpg?imageView2/2/w/200

Github + Hexo 方便好用

Github、Hexo、Markdown、域名绑定 折腾了一早上，终于搞定了。

生成网站

hexo g == hexo generate

发布网站

hexo d == hexo deploy

测试网站

hexo s == hexo server

发布文章

hexo n == hexo new

多重命令

hexo s -g
