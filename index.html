蜜蜡

蜜蜡即密腊，是琥珀的一个品种。呈不透明状或半不透明状的琥珀被称作密腊，树木脂液化石，为非晶质体。组织较疏松而又不含其他矿物的蜜蜡能浮于水（英文蜜蜡Amber一词自阿拉伯文，其原义即为“浮于水”的意思）。

　　蜜蜡的颜色有蛋清色、米色、浅黄色、鸡油黄，橘黄色等黄色系为主的颜色。国内最常见的蜜蜡主要是产自波罗的海、缅甸血珀、墨西哥跟多米尼加的蓝珀，其中以波罗的海的乌料与俄料最为常见。

　　以常见的波罗的海琥珀为例，完全透的是金珀，不透的是蜜蜡。当然之间还有半蜜半珀的状态。

　　蜜蜡蕴含无数的色彩，有的透明晶亮，有的半透明，有的不透明但色纹斑斓。透明的若再加上光线照射，往往有多种色彩显现。自古以来，蜜蜡便为世人所喜爱，且人不分疆界、种族、阶级、文化、宗教和时代背景，均对之赞赏有加，视同宝物，历久不衰。

　　蜜蜡象征永恒的爱侣，不断的散发莫名魅力，愿每天为它写下千百首赞美情诗，来表达对它那份热切追求的心意，并全心全意全力的去爱惜，无论置身于哪一个时空中，蜜蜡之美名就如它本身一样的纯洁美丽，与一样的完美。

2015-11-19 使用7牛图片存储

  • Github 空间有1G大小限制，另外Github服务器在国外, 考虑速度，将图片进行分流。

  • 目前尝试使用 7牛 www.qiniu.com 的图片存储。7牛免费资源 (10GB永久免费存储空间/每月10GB下载流量/每月10万次Put请求/每月100万次Get请求) 目前应该是够用了。

  • 关于图片的发布，7牛的同步工具 qrsbox 也挺好用的，可以自动监控同步文件图片，上传到服务器。7牛服务器没有目录，同步后链接前缀+图片文件名就可以直接访问了。

  • 关于图片的使用，赞一个的是7牛提供的图片外链支持参数，可以调整图片大小，弥补了Markdown不能定义图片大小的问题。

七牛图片基本处理（imageView2）
